## flame - 1.0 
## a simple program to help you get started with python!

**for debian based linux distros only!**

## how to use

1. clone this git repo

2. find the flame
x86_64 executeable

3. type chmod +x flame
in a terminal or right click
then find and check "allow program
to run as executeable"

4. left click the executeable or
in a terminal type "./flame"
